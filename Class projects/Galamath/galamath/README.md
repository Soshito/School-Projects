# GalaMath



## Getting started
### First: Clone the project to your device.
### Second: Install the dependencies, random, time, and pygame.
### Third: For the Game images and sounds to work you must have the assets folder in the same directory as the main GalaMath file.
### Fourth: Run the project

## How To Play
### Movement: Move your ship using the 4 arrow keys
### Shooting: Your ship will shoot once every time you hit the space bar (given you have ammo left)
### Scoring: You will gain 100 points for each enemy ship destroyed and 1000 points for each math asteroid correctly destroyed
### Difficulty (starting HP): Easy - 3, Medium - 2, Hard - 1
### Damage: if you are hit by an enemy ship's bullet you will lose 1 HP. If you lose all of your HP the game will end and your score will be saved if you have the new high score
